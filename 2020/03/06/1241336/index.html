前言

因为疫情的原因,导致在家呆了两个月,也学习了两个月,在tomcat 的webapps目录下塞满了war包

以下来介绍一下我的学习成果

Perfume商店

香水商店,经过一次一次的升级,有三个版本,是技术推进的结果.

三个版本概述:

  1. perfume是直接在jsp之中循环展示商品;
  2. perfume-2是新增了购物车的功能,这里学习到了ajax 局部刷新技术;
  3. perfume-3使用JSTL来简化繁琐的循环和数据的获取

链接:

  1. http://47.115.33.215/perfume/
  2. http://47.115.33.215/perfume-2/
  3. http://47.115.33.215/perfume-3/

主页

这个主页原本的打算是用来当个人博客用的,在后期写博客的时候发现了局限性,以至于博客迁移在这里.

链接: http://47.115.33.215/

武汉加油项目

在疫情的之下,让我们为武汉人民加油,武汉加油,中国加油!

链接: http://47.115.33.215/fighting-wuhan/

水果商店

总体来说跟perfume商店是差不多的,使用到了Modal跟Dao层,渐渐有MVC的味道了.

链接: http://47.115.33.215/fruitProj2/

生活商城

这是最近这次实验报告的商城,有登录功能等,主要是用来理解request和response的功能和奥义.

链接: http://47.115.33.215/market/index.jsp

PicGo + Github 实现图床

微博图床无法使用之后,写博客的一片哀嚎.

使用PicGO + Github 就是一套解决方案,中间采用了GitHub的CDN加速,避免了访问速度慢的缺点.

学习Linux

自从领了新的云服务器,就去搞了去Ubuntu系统,在cmd式的交互下,安装tomcat,mysql等一些东西,还是有点快乐的.

总结

以上便是这两个月的总结，学的不多，继续努力！fighting！
